# SplitSmart

SplitSmart is a backend API project for managing shared group expenses. It allows users to create groups, add members, record expenses, calculate balances, and generate settlement suggestions.

## Current Features

- Group CRUD APIs
- Group member management
- Expense creation with validation
- Expense retrieval by group
- Balance calculation
- Settlement calculation

## Tech Stack

- ASP.NET Core Web API
- C#
- Minimal APIs
- Swagger
- Git and GitHub

## Project Status

Backend MVP in progress.
